LuLu Shopping

Final Project at Eduwork

  • Rating:
  • Client: Lulu
  • Website: LuLu
  • Completed: 13 Apr 2023
  • Repository : Github

In this final project I made a test case on Real Product using the LuLu Shopping application which was downloaded via apk.pure and used a Pixel 5 device as an emulator with Android OS 12 The test case being tested is checking out an item in the LuLu Shopping application as a Guest using Katalon Studio as an Automated testing tool.

Scenario : checkout as guest

Steps :

1. Start application

2. tap the 'SKIP' button

3. select the language 'English'

4. click 'Continue'

5. click 'Get started'

6. Because you act as a guest, please click the skip button to create an account

7. Enter 'delivery mode', select the 'Home Delivery' option

8. input the address of the message recipient

9. click 'confirm location'

10. If you have entered the homepage, please select the item you want to order

11. click add item

12. then enter the Cart

13. Please confirm the order

14. then click 'proceed'

15. then enter your email address and click 'Proceed'

16. then please fill in the form for a new account and after that click 'Continue'

17. There will be a verification code that will enter your email

18. Done
